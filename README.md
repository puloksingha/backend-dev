# backend-dev

A learning repository for backend development using **Node.js** (JavaScript). The project is organized as day-by-day practice notes and code examples.

## Repository structure

- `node-js/`
  - `Day-01/` — Intro to Node.js (notes)
  - `Day-02/` — Core modules + npm practice (code + notes)
  - `Day-03/` — (practice)
  - `Day-04/` — (practice)

## Getting started

### Prerequisites

- Node.js (LTS recommended)
- npm (comes with Node.js)

### Run the Day 02 demo

Day 02 contains a small script demonstrating:

- OS info via Node's `os` module
- Simple math functions
- File read/write using `fs` + path utilities

```bash
cd node-js/Day-02
npm install
npm start
```
